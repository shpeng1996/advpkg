# [News] Samsung Reportedly Targets 2026 CXL 3.2 Mass Production; SK hynix Advances New AI Memory Architecture

Source: TrendForce, Seoul Economic Daily, BigGo Finance | Date: 2026-07-21

## Samsung CXL 3.2 Mass Production

- **Samsung CMM-D (CXL Memory Module-DRAM) 3.0** based on CXL 3.2 specification: mass production **by end of 2026**
- CXL memory can increase total memory capacity by **up to 50%** and **double memory bandwidth** compared to DDR5-only systems
- Samsung and SK hynix are racing to mass-produce CXL 3.2, signaling a new memory tier era

## SK hynix IMTE Architecture

- SK hynix introduced **IMTE (Inference Memory Tiering Expansion)** architecture
- Improves AI inference efficiency by **+35.7%** over conventional systems
- Architecture: places **CXL hybrid memory** between HBM/DDR and SSD — a new "warm memory" tier
- SK hynix showcased second-generation **CMM-DDR5** (CXL 3.2-based) samples at HPED 2026

## Strategic Context: Beyond HBM

- As HBM and DDR approach capacity limits for AI workloads, memory makers are creating new CXL memory tiers
- CXL memory expands capacity at lower cost without adding more GPUs/CPUs
- Both Samsung and SK hynix positioning CXL as "next memory" — complementary to (not replacing) HBM
- New tier hierarchy: HBM (highest bandwidth/cost) → DDR5 → CXL → SSD

## Packaging Implications

- CXL memory uses standard DDR5 packaging (no advanced interposer required)
- Contrasts with HBM's tight coupling with CoWoS interposer
- Creates a new market segment for AI memory that bypasses advanced packaging bottlenecks
