JEDEC published the SPHBM4 standard (JESD330-4) — Standard Package High Bandwidth Memory — which combines HBM4 DRAM ICs with standard packaging and a fast 'narrow' 512-bit interface.

## Key Technical Details
- **Standard**: JESD330-4 (SPHBM4 = Standard Package HBM4)
- **Interface**: 512-bit narrow interface (vs. standard HBM4 1024-bit wide interface)
- **Substrate**: Organic package substrates replace expensive silicon interposers
- **Bandwidth class**: HBM4-class bandwidth despite narrower interface
- **Cost reduction**: Eliminates the need for costly silicon interposers (CoWoS)

## Significance
- SPHBM4 enables HBM4-class memory performance at lower cost by using organic substrates instead of silicon interposers
- The standard allows chipmakers to implement high-bandwidth memory without the premium cost of CoWoS or EMIB-based 2.5D packaging
- This is NOT a GDDR killer — it occupies a different tier between commodity DRAM and full HBM4
- Named participants in JEDEC working group include SK Hynix, Samsung, Micron, and Chinese DRAM maker CXMT

## Market Implications for Advanced Packaging
- Could reduce demand for silicon interposer-based 2.5D packaging (CoWoS) for some use cases
- Addresses mid-tier AI accelerator and edge AI applications where CoWoS is cost-prohibitive
- Similar positioning to Intel XBM (UCIe 32 GT/s approach) as an alternative to traditional HBM interposer architecture
