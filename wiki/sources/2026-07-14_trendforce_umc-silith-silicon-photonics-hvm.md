# UMC + SILITH 矽光子 HVM 首批出貨 / UMC + SILITH Silicon Photonics First HVM Delivery

## 核心主張 / Key Claims

- **里程碑**：UMC 新加坡 12 吋廠正式開始量產矽光子晶圓，標誌矽光子從開發到高量產的轉折點
- **客戶鏈**：SILITH → Innolight/Coherent（光收發器）→ NVIDIA/Google；已通過「leading cloud infrastructure customer」量產資格認證
- **規格**：200G/lane 矽光子，路線圖延伸至 400G/lane
- **UMC 封裝服務**：2027 年提供先進封裝服務（photonic chip + interposer），直接進入 CPO 供應鏈
- **市場預測**：TrendForce CPO/NPO 市場 >$39B by 2030，2028-2029 加速

| 指標 | 數值 |
|------|------|
