# NVIDIA Rosa CPU 評估採用 TSMC A16 / NVIDIA Rosa CPU May Adopt TSMC A16 with Back-Side Power Delivery

## 核心主張 / Key Claims

- NVIDIA 下一代 **Rosa CPU** 評估採用 TSMC **A16（含 Super Power Rail 背面供電）**
- Rosa + Feynman 資料中心平台：**2028 年**推出
- Rosa 採用 **Rigel 核心**（Arm v9.2）：128+ 核心，IPC 預期 **+50% vs Vera CPU**
- A16 SPR 使 CMP 需求在 2nm 基礎上再增 **15–20%**；**載板晶圓 ASP >4x 再生晶圓**
- Vera CPU 首批確認客戶：Perplexity、OpenAI、Anthropic、Oracle

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| A16 vs N2P 密度提升 | 1.10× |
| A16 CMP 步驟增加 | +15–20% vs 2nm |
| 載板晶圓 ASP | >4× 再生晶圓 |
| Rosa Rigel 核心數 | 128+（預估）|
| IPC 改進 | >50% vs Vera/Olympus |
| 資料中心平台推出 | 2028 |
| 消費級平台（Spark） | 2030 |

## 新增知識 / New Knowledge Added

1. **NVIDIA CPU 路線圖延伸至 2028–2030**：Rosa（A16）→ 2028；Spark（消費級）→ 2030，填補 Vera（2026）之後的路線圖空白
2. **A16 SPR 對封裝材料的影響**：CMP 需求增加 15–20%，載板晶圓 ASP >4x 再生晶圓——這是 A16 採用對封裝供應鏈影響的首次量化
3. **Vera CPU 生態系客戶確認**：Perplexity + OpenAI + Anthropic + Oracle 均使用 Vera CPU，確認 TSMC CoWoS 需求將隨 CPU 擴散
