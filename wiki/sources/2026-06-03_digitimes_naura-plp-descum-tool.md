# Naura 推進 AI 晶片封裝：首批 600mm PLP 去膠渣設備出貨 / Naura pushes into AI chip packaging with first 600mm PLP descum tool

**注意**：文章有訂閱牆，以下內容基於頁面 metadata 與標題摘錄。

## 核心主張 / Key Claims

1. 中國半導體設備龍頭**北方華創（Naura Technology Group）**已出貨首台 **600mm × 600mm 面板級封裝（PLP）去膠渣設備**
2. 此舉標誌 Naura 從晶圓級封裝（WLP）設備擴展至面板級封裝（PLP）設備
3. Naura 原為前端製程（刻蝕、CVD）設備主力廠商，進入 PLP 設備市場是多元化佈局的一步

## 關鍵數據 / Key Data Points

| 項目 | 數據 |
|------|------|
| 面板尺寸 | 600mm × 600mm |
| 工具類型 | 去膠渣（descum）設備 |
| 廠商 | Naura Technology Group（北方華創） |
| 出貨時間 | 2026-06-03 報導 |

## 新增知識 / New Knowledge Added

- **中國 PLP 設備自主化**：Naura 是中國半導體設備的標誌性廠商（已在刻蝕、CVD、爐管等前端製程佔有重要市場），其進入 PLP 設備市場代表中國先進封裝設備生態正在加速形成。
- **600mm 面板格式**：600mm × 600mm 面板規格與 Rapidus 玻璃基板樣品尺寸（600×600mm 最大樣品）、USHIO 510mm 面板一致，顯示業界正朝更大面板格式推進。
- **設備競爭新玩家**：在 AMAT（收購 NEXX）、TEL、Lam Research 大舉進入封裝設備市場的同時，Naura 的加入顯示中國本土廠商也在追趕面板封裝設備機會。

## 矛盾或修正 / Contradictions / Corrections

- 無矛盾。現有 wiki（FOPLP 頁面）已記錄 USHIO 510mm、ASE 310mm 等設備進展，Naura 600mm 是補充中國設備廠商視角。
