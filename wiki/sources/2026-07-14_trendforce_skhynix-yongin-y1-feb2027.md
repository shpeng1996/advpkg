# SK Hynix Yongin Y1 設備訂購啟動 / SK Hynix Yongin Y1 Equipment Orders

## 核心主張 / Key Claims

- **時程提前 3 個月**：Y1 Phase 1 由原計畫 2027-05 提前至 **2027-02**；設備安裝 2027-03~04
- **初始產能**：~20,000 wsm，聚焦 1c（第六代 10nm 級）DRAM
- **產品用途**：DDR、LPDDR、**HBM4E**（最快 2027 年商用）
- **Phase 2+3**：潔淨室建設 2H 2026 啟動
- **龍仁叢集完成時程**：4 座廠 2033（原 2045，提前 12 年）
- **三星龍仁**：首座廠 2029（前進代工，提前 2 年）

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| Y1 Phase 1 開幕 | 2027-02（提前自 2027-05） |
| 設備安裝斜坡 | 2027-03~04 |
| 初始產能 | ~20,000 wsm |
| 製程節點 | 1c DRAM（第六代 10nm 級） |
| 龍仁叢集完成 | 2033（原 2045） |
| 三星龍仁第一廠 | 2029（代工，原 2030-31） |

## 新增知識 / New Knowledge Added

SK Hynix Yongin Y1 Phase 1 精確開幕時程首次量化為 2027-02（上一個已知時程為「2027 年初」）。設備訂購已正式啟動（2026-07），提供了 HBM4E 2027 年商用化的基礎設施支撐。
