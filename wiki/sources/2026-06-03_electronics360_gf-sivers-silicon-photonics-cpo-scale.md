# GlobalFoundries + Sivers 矽光子 CPO 合作（SCALE 平台）/ GF + Sivers Silicon Photonics CPO (SCALE Platform)

## 核心主張 / Key Claims

1. **GF + Sivers Semiconductors AB 戰略合作**：瑞典光子器件供應商 Sivers 的雷射陣列將整合至 GF 的矽光子平台參考設計中
2. **GF 是全球最大純晶圓代工廠矽光子代工廠**（收購新加坡 Advanced Micro Foundry，AMF 後）
3. **矽的根本限制**：矽無法自行發光，需要外部雷射光源——Sivers 填補此缺口
4. **SCALE 平台完整光學連接架構**：支援插拔式收發器（pluggable）→ 線性插拔光學（LPO）→ 共封裝光學（CPO）三個市場段
5. **CPO 定位為「下一個前沿」**（Next Frontier）——明確確認 GF 的 CPO 商業化路線
