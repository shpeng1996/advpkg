# COMPUTEX 2026 預覽：CPO 浮現主舞台 / COMPUTEX 2026 Preview: CPO as a Key Theme

## 核心主張 / Key Claims

- COMPUTEX 2026（6/2–5）CPO 與 AI 伺服器為核心展示主題。
- **Wiwynn + Ayar Labs**：將展示最新 CPO 光學互連技術。
- **Largan Precision**（蘋果鏡頭供應商）：COMPUTEX 首次亮相，展示用於**光纖陣列單元（FAU）的玻璃元件**，凸顯 CPO 供應鏈多元化。
- **MediaTek**：展示 **400Gbps-per-fiber CPO 技術** + MicroLED 光學解決方案（AOC 應用）；相對銅線功耗降低 50%。
- NVIDIA RTX Spark 發表（GTC Taipei 6/1）：N1X Arm CPU（與 MediaTek 共同設計，TSMC 3nm）。

## 關鍵數據 / Key Data Points

| 主題 | 規格/數據 |
|------|---------|
| MediaTek CPO 頻寬 | 400Gbps/fiber |
| MediaTek AOC 功耗改善 | -50% vs 銅線 |
| Vera Rubin 首批出貨 | 2H26（Q3 開始） |
| NVIDIA RTX Spark N1X 製程 | TSMC 3nm |
| 零件短缺緩解預期 | 2027 年底（Wiwynn 預估） |

## 新增知識 / New Knowledge Added

- **MediaTek 400Gbps CPO 技術首次進入 wiki**：wiki 先前記錄 MediaTek 雙封裝策略（EMIB + CoWoS），未記錄其 CPO 能力。此文確認 MediaTek 已開發自主 CPO 光學互連，且功耗優於銅線 50%。
- **Largan 進入 CPO 供應鏈（FAU 玻璃元件）**：Largan 的 COMPUTEX 首秀確認其轉型 CPO 供應鏈的策略，wiki 首次記錄。
