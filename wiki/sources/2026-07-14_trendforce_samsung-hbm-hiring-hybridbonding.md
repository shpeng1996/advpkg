# Samsung HBM 全製程招募 / Samsung HBM Full-Process Hiring

## 核心主張 / Key Claims

- **招募範圍**：6 個 HBM 崗位，涵蓋 core die 設計、base die 設計、可靠性評估、封裝開發、應用工程、封裝製程開發（半導體研究中心）
- **時程**：2026-07-13 至 2026-07-27
- **混合接合 + HPB（Heat Path Block）**：Samsung 明確表示計畫將混合接合與 HPB 共同推進，用於高層數 HBM 產品的封裝減薄與熱阻降低
- **封裝製程職能**：multi-stack hybrid copper bonding、TSV 模組開發、HBM copper pad 平坦化
- **SK Hynix 同步招募**：HBM Foundry Process Integration + HBM Digital Design（2026-06-24）

## 關鍵數據 / Key Data Points

| HBM 代際 | 現狀 |
|----------|------|
| HBM4（6th gen） | 量產進行中 |
| HBM4E（7th gen） | 客戶樣品送樣中（2026） |
| HBM5（8th gen） | 技術發布完成 |

## 新增知識 / New Knowledge Added

1. **Samsung HPB（Heat Path Block）明確與混合接合結合**：本次招募明確將「hybrid bonding + HPB」定義為高層數 HBM 的封裝路線，補充了 wiki/technologies/hybrid-bonding.md 中 Samsung HPB 與接合技術的整合路線
2. **SK Hynix 「HBM Foundry Process Integration」職缺**：首次明確記錄 SK Hynix 正在組建與代工廠的整合能力，印證其 HBM base die 委外台積電策略需要內部 PI 能力
