# POSTECH 十層晶片堆疊技術：達 12-Hi HBM 四倍密度

## 核心主張 / Key Claims

1. **POSTECH（浦項科技大學）研發出「單步驟轉移接合」製程**，可在低溫低壓環境（< 180°C、< 20 kPa）下實現超薄晶片的穩定多層堆疊，突破現有 HBM 堆疊的物理密度上限。
2. **10 層以上晶片穩定堆疊成功驗證**，且達到 12-Hi HBM 標準堆疊密度的**四倍**，具重大學術突破意義。
3. **關鍵技術：transfer + bond 單步同步完成**（相對傳統需分離進行的多道工序），由 Kim Seok 教授團隊研發；論文發表於 *Results in Engineering* 期刊。
4. **晶片尺寸**：14 µm 超薄晶片（Die thickness）為主要測試物件，說明此技術在 TSV 後薄化晶粒上的應用潛力。
5. 研究團隊亦驗證 Micro-LED 陣列的多層堆疊可行性，顯示此技術跨越記憶體、顯示器等多個應用領域。

## 關鍵數據 / Key Data Points

| 參數 | 數值 |
|------|------|
| 堆疊層數 | 10+ 層（穩定驗證） |
| 面積密度 | 12-Hi HBM 的 4 倍 |
| 製程溫度 | < 180°C |
| 製程壓力 | < 20 kPa |
| 晶片厚度（測試件） | 14 µm |
| 接合步驟 | 單步驟（transfer + bond 同步） |
| 論文期刊 | Results in Engineering |
| 主責研究員 | Prof. Kim Seok（POSTECH） |

## 新增知識 / New Knowledge Added

- **首次收錄 POSTECH 相關研究**：此前 wiki 無 POSTECH 記錄。本文顯示韓國頂尖理工大學正積極研究 HBM 後繼堆疊技術，且具體量化優勢（4×密度）超越現有商業產品。
- **低溫低壓接合工藝的突破性意義**：< 180°C 代表無需高溫回流（reflow），可降低晶片翹曲（warpage）與熱應力；< 20 kPa 低壓代表薄晶片破片風險大幅下降。這兩點是現有 TC-NCF/MR-MUF 混合接合仍需克服的痛點。
- 此研究提供了一條**獨立於 TSMC SoIC 和 Samsung X-Cube 之外的學術技術路線**，為後 HBM4 時代的記憶體堆疊架構競賽注入新可能。

## 矛盾或修正 / Contradictions / Corrections

學術原型（lab-scale）與量產（mass production）仍有相當距離。此技術目前**不宜直接與 SK Hynix/三星 HBM 量產路線比較**，應視為 5–10 年後商用化的技術儲備。
