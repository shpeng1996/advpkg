# 三星申請 HBM 虛擬晶片（Dummy Die）新專利，瞄準 16 層 HBM5 良率

## 核心主張 / Key Claims

1. 三星針對高堆疊 HBM 的可靠性問題申請新專利：虛擬晶片（dummy die）側壁設計為**三段式梯形結構（3-step terraced）+ 曲面輪廓**，結合現有混合接合（hybrid bonding）與 HPB（Heat Path Block）技術。
2. **目標用途**：HBM5（16 層以上）——12 層以上堆疊中，頂層虛擬晶片翹曲（warpage）是製造良率的關鍵制限因素。
3. **良率背景**：8 層→12 層 HBM 良率下降 10–20 個百分點；16 層 HBM5 良率估計僅 40–60%。
4. **製程創新**：深溝槽鋸切（Deep Groove Sawing，雷射分割）取代機械刀切，精度更高、晶格損傷更少。
5. **散熱設計**：接合絕緣層底面與水平延伸面間保留 1–10 µm 垂直距離以維持導熱效率；突出表面設計減少 EMC（環氧模封料）體積以改善散熱。

## 關鍵數據 / Key Data Points

| 參數 | 數值 / 說明 |
|------|------------|
| 目標產品 | HBM5，16 層以上堆疊 |
| 16 層 HBM 估計良率 | 40–60%（行業估計） |
| 8→12 層良率降幅 | 10–20 個百分點 |
| 散熱接合距離 | 1–10 µm（鍵合絕緣層底部至水平延伸面） |
| 製程 | Deep Groove Sawing（雷射分割）+ 倒錐形輪廓 |
| 結合技術 | Hybrid Bonding + HPB |
