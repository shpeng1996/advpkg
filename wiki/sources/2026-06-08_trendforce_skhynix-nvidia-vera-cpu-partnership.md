# SK hynix 多年期 NVIDIA 記憶體夥伴關係擴大 / SK hynix Multi-Year NVIDIA Memory Partnership Expands

## 核心主張 / Key Claims

- SK hynix 與 NVIDIA 簽署多年期記憶體夥伴關係協議（2026-06-08），涵蓋 Vera Rubin AI 超算、Vera CPU、RTX Spark PC 及 Jetson Thor 機器人平台
- NVIDIA Vera CPU 採用第二代 **LPDDR5X**，頻寬高達 **1.2 TB/s**（傳統 CPU 設計的 2 倍）；記憶體功耗低於 **30W**（DDR5 系統超過 100W）
- SK hynix 確認為 NVIDIA 最大記憶體供應商，但協議並非獨家授權（multi-vendor 策略不變）
- Samsung 亦通過 HBM4 認證、已獲量產供應資格，黃仁勳同日與三星 DS 事業部 Jun Young-hyun 副會長會面

## 關鍵數據 / Key Data Points

| 指標 | 數值 | 備註 |
|------|------|------|
| Vera CPU 記憶體頻寬 | 1.2 TB/s | 傳統 CPU 設計 2 倍 |
| Vera CPU 記憶體功耗 | <30W | vs DDR5 系統 >100W |
| LPDDR5X 世代 | 第二代 | 能耗/bit 顯著低於 DDR5 |
| 協議性質 | 多年期（multi-year） | 涵蓋 Vera Rubin AI、Vera CPU、RTX Spark、Jetson Thor |

## 新增知識 / New Knowledge Added

- **NVIDIA Vera CPU 首次明確記憶體規格**：1.2 TB/s / LPDDR5X / <30W，為 wiki「agents AI 時代 CPU 崛起」論述補上技術規格佐證
- **SK Hynix 夥伴關係從 GPU 端擴展到 CPU 端**：不再只是 HBM4 供應商，而是整個 NVIDIA AI 基礎架構的全線記憶體夥伴
- **Multi-vendor 策略再確認**：SK Hynix 無獨家地位；Vera Rubin 和 Vera CPU 均採 SK Hynix + Samsung + Micron 多源
