# SK hynix Completes 12-High Hybrid Bonding HBM Validation / SK Hynix 完成 12 層混合接合 HBM 驗證

## 核心主張 / Key Claims

- SK hynix Technical Leader Kim Jong-hoon（金鐘勳）確認 12-high HBM 混合接合驗證已完成，正致力提升量產良率
- 公司表示「比兩年前更有充分準備」，但未揭露具體良率數字
- 業界預期混合接合從 2026 H2 至 2027 年隨 16-high 產品商業化逐步引入
- SK Hynix 完成首批混合接合量產設備採購：向 Applied Materials + Besi 訂購 inline 系統，總價約 KRW 200 億（~USD 1,500 萬）
- 三星也引入 Besi 混合接合設備用於開發，並以 SEMES 工具進行品質測試（但 SEMES 成熟度較低）

## 新增知識 / New Knowledge Added

1. **里程碑確認**：SK Hynix 官方確認完成 12-high HBM 混合接合驗證，是業界首個具名技術負責人公開確認的里程碑（Kim Jong-hoon, Technical Leader）
2. **首批量產設備採購**：AMAT+Besi inline 混合接合系統訂單——標誌 SK Hynix 從研發轉向量產規劃的正式開始
3. **三星同步行動**：三星也在測試 Besi 及 SEMES 混合接合設備，但 SEMES 平台成熟度不足

## 矛盾或修正 / Contradictions / Corrections

無重大矛盾。與 2026-01-13 文章（SK Hynix 維持 MR-MUF 至 16H）一致：12H 驗證完成 ≠ 立即量產；MR-MUF 仍是當前 16H 主力，混合接合是下一代。
