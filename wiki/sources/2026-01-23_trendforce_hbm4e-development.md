# HBM4E 開發時程：三大廠商並行、2027 年佔市場 40%

## 核心主張 / Key Claims

1. **Samsung** 客製化 HBM4E 設計目標完成時間：**2026 年中**
2. **SK Hynix** 同樣鎖定 HBM4E 開發完成於 **2026 年上半年**
3. **Micron** HBM4E 計畫於 **2027 年**推出，與 TSMC 合作開發
4. HBM4E 預估將佔 2027 年 HBM 總需求的 **約 40%**
5. **HBM5** 預計 **2029 年**問世

## 關鍵數據 / Key Data Points

| 規格 | HBM4 | HBM4E |
|------|------|-------|
| 資料速率 | 6.40 GT/s | 10 GT/s |
| 頻寬/堆疊 | ~1.65+ TB/s | 2.5 TB/s |
| 介面寬度 | 2048-bit | 2048-bit |
| 每封裝功耗上限 | ~60 W | 80 W |
| 堆疊高度 | 12-Hi (標準) | 16-Hi (可能) |
| 量產啟動（Samsung） | 2026-02 | 2027 目標 |
| 量產啟動（SK Hynix） | 2026-H1 | 2026-H1 開發完成 |
| 量產啟動（Micron） | 2026 | 2027 |

## 新增知識 / New Knowledge Added
