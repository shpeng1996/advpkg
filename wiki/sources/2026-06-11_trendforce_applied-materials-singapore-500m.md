# Applied Materials 開設 5 億美元新加坡廠 / Applied Materials Opens $500M Singapore Site

## 核心主張 / Key Claims

- **$500M 新加坡製造園區開幕**：AMAT 在新加坡新增製造基地，將當地廠數從 1 座增至 3 座，新加坡承擔全球產能約一半。
- **半導體設備業務 2026 年成長 >30%**，2027 年後需求維持強勁。
- **先進封裝設備收入 2026 年預計增長 50%**（Gary Dickerson 明確表示）。
- **WFE 成長中 >80% 與先進邏輯、記憶體及先進封裝相關**——封裝已成為 WFE 成長的主要驅動力之一。
- **AI 資料中心基礎設施投資：4-5 年累計 $7 兆**（AMAT 預測）。
- **8 季滾動預測**：主要客戶（TSMC/Intel/Samsung/SK Hynix/Micron）提供前所未有的供應鏈可視度。

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| 新加坡廠房投資額 | US$5 億 |
| 新加坡廠數 | 1 → 3 座（約佔全球產能一半） |
| 2026 年半導體設備業務成長預測 | >30% |
| 2026 年先進封裝設備收入成長 | **+50%** |
| WFE 成長中先進領域佔比 | >80% |
| AI 資料中心 4-5 年投資預測 | US$7 兆 |
| 新加坡員工數 | ~4,000 → +1,000（計畫） |

## 新增知識 / New Knowledge Added

- **AMAT 先進封裝設備 +50% 2026 年**：比此前已知的 >50% 成長描述更具體，CEO 親口確認。
- **WFE >80% 集中在先進領域（含先進封裝）**：量化了先進封裝在整體設備需求中的相對重要性，確認「後端正在追上前端」。
- **8 季預測共享**：客戶向 AMAT 等設備商提供超長滾動預測，是 AI 週期與傳統半導體週期的重要結構差異。

## 矛盾或修正 / Contradictions / Corrections
