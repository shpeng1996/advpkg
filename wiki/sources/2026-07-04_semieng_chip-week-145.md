# Chip Industry Week in Review #145 — 記憶體設備支出突破 $50B / Memory Equipment Spend Surpasses $50B

## 核心主張 / Key Claims

- 2026 年 300mm 記憶體晶圓廠設備支出首次突破 **$50B**，年增 29%；2027 預估再增 11%
- DRAM 設備：$37B；3D NAND 設備：$14B
- 前十大晶圓代工廠 8 吋產能利用率回升至 **88%**
- 代工廠價格 Q1→Q2 漲幅 **5–15%**
- Nvidia 推出 AI 新創收益分潤（revenue-sharing）算力方案
- 80 家新創在 AI/EDA/製造領域合計募資 **$8.4B**

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| 300mm 記憶體設備支出（2026） | >$50B（+29% YoY） |
| DRAM 設備 | $37B |
| 3D NAND 設備 | $14B |
| 2027 記憶體設備預測 | +11% |
| 8 吋代工利用率 | 88% |
| 代工價格漲幅 | Q1→Q2 +5–15% |
| AI/EDA/製造新創募資 | $8.4B（80 家） |

## 新增知識 / New Knowledge Added

- $50B 記憶體設備歷史首破，量化 HBM 驅動的設備端規模（$37B DRAM 佔 74%）
- Nvidia revenue-sharing 算力模式為 AI 新創算力取得開創新路徑
- 代工廠價格上漲 5–15% 確認先進封裝服務商毛利改善趨勢
