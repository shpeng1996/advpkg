# Intel 正式加入 TeraFab 計畫，強調封裝能力 / Intel Officially Joins TeraFab, Highlights Packaging

## 核心主張 / Key Claims

- Intel 於 2026-04-07 正式宣布加入 TeraFab 計畫，成員包含 SpaceX、xAI、Tesla
- Intel 官方聲明明確點名「封裝」為核心貢獻能力：「設計、製造、**封裝**超高性能晶片的能力」
- TeraFab 目標：每年產出 1 TW 算力，定位為「有史以來最史詩的晶片製造計畫」
- 訊息透過 X 平台發布，無正式新聞稿或 SEC 申報
- Lip-Bu Tan（Intel CEO）公開背書：「Elon 有重新定義整個產業的實績」

## 關鍵數據 / Key Data Points

- TeraFab 目標產能：**1 TW/year**（1 兆瓦/年算力）
- 成員：SpaceX + xAI + Tesla + Intel（截至 2026-04）
- 宣布方式：X 平台貼文（非正式聲明）
- Elon Musk 同週末訪問 Intel 設施

## 新增知識 / New Knowledge Added

- **首次明確記錄 Intel 加入 TeraFab**，且 Intel 將先進封裝（EMIB/Foveros）列為核心參與理由
- TeraFab = 邏輯 + 記憶體 + 先進封裝三位一體大規模「重構」的新型態聯盟
- 為 Intel Foundry 提供 xAI/SpaceX/Tesla 等非傳統客戶管道
