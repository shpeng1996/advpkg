# EV Group ECTC 2026：450nm Cu-Cu HB 98% 良率；300nm 混合接合；D2W 計量（Intel）；RDL 面板

## 核心主張 / Key Claims

1. ECTC 2026（2026-05-26~29，奧蘭多）：EVG 展示 GEMINI FB（W2W）、EVG40 D2W（計量）、IR LayerRelease（層轉移）、LITHOSCALE XT（無光罩微影）
2. **Session 18（EVG + Applied Materials）**：**450nm pitch Cu-Cu 混合接合，98% 良率，橫跨 2,000 萬個互連**——業界首次 <500nm pitch W2W 高良率示範
3. **Session 26（EVG + Applied Materials）**：**300nm pitch W2W 混合接合**，SiCN 介電層，50nm overlay，細晶粒銅冶金，含可靠性評估
4. **Session 39（EVG + Intel）**：D2W 高速 100% 接合 Overlay 量測——Intel 共同作者，顯示 Intel 在 D2W HB 路線的設備選擇傾向
5. **Session 27（EVG + Asahi Kasei）**：310×310mm² 面板基板無光罩微影用於高深寬比 Cu Pillar——FOPLP 面板關鍵製程

## 關鍵數據 / Key Data Points

| 指標 | 數值 | 備註 |
|------|------|------|
| W2W HB pitch（新） | 450nm | EVG + AMAT；Session 18 |
| W2W HB 良率 | 98% | 橫跨 20M 互連 |
| W2W HB pitch（窄） | 300nm | EVG + AMAT；Session 26 |
| D2W overlay 精度 | 50nm（300nm pitch） | Session 26 |
| Panel substrate size | 310×310mm² | Session 27，FOPLP |
| LITHOSCALE XT 通量 | 5× vs. 前代 | 高通量無光罩 |

## 新增知識 / New Knowledge Added

- **450nm pitch Cu-Cu W2W HB 首次示範（ECTC 2026）**：這是目前 wiki 中記錄的最高密度 W2W 接合數據，超越先前已知的 1µm（CEA-Leti D2W）。W2W 場景下 450nm 更高密度，98% 良率橫跨 20M 互連——填補了「大面積 W2W 高良率」的知識空缺。
- **300nm pitch W2W HB（Session 26）**：EVG + AMAT 合作的第二篇論文，進一步深化至 300nm，配備 SiCN 介電層（與低溫製程相容）——與 wiki 現有低溫路線敘述形成互補。
- **Intel 共同作者 D2W 計量（Session 39）**：Intel 與 EVG 合作開發 D2W 高速 overlay 量測，顯示 Intel 在 D2W HB 路線上依賴 GEMINI/EVG40 系統。
- **EVG IR LayerRelease**：可消除某些封裝流中的玻璃基板需求，以矽晶圓為臨時載體——對 FOPLP 玻璃基板路線可能是補充替代方案。
- **Ruthenium 金屬接合（Session 33）**：EVG + imec：Epitaxial Ru 層轉移，金屬-金屬接合，走向單晶互連——屬於前沿探索（下一代互連材料研究方向）。
