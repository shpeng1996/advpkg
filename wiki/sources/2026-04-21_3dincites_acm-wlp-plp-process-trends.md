# WLP/PLP 製程趨勢：前端嚴格度向後端遷移 / WLP/PLP Process Trends

## 核心主張 / Key Claims

- WLP（晶圓級封裝）的核心驅動力已從封裝設計轉向**高精度製程技術**（ECP、PECVD、濕製程）
- PLP（面板級封裝）正成為 WLP 的補充規模化路徑，但引入更高的均一性、材料處理與製程整合挑戰
- 高深寬比 TSV、超細 RDL、混合接合等技術要求**前端製程嚴格度**在後端/OSAT 製造中實現
- 永續性（化學品回收、封閉循環）正成為 HVM 的平行驅動力，不只是技術性能
- **前端設備廠商**（ACM Research、Lam Research、AMAT、TEL）正大舉進入先進封裝製程設備市場

## 關鍵數據 / Key Data Points

| 製程 | 先進封裝中的角色 |
|------|----------------|
| ECP（電化學鍍銅） | Cu bump 形成、RDL 金屬填充、TSV 填孔 |
| PECVD | 共形介電質沉積、鈍化層 |
| 濕製程 | 表面準備、塗布、顯影、剝離、蝕刻 |
| 混合接合前處理 | 表面平坦化、Cu CMP、污染物控制 |

## 新增知識 / New Knowledge Added

**設備供應鏈重整框架**（ACM Research 視角）：
- 2000年代：OSAT 專屬工具，製程容差較寬鬆
- 2024年後：OSAT 工具已無法滿足超細間距要求 → 前端設備廠商進入後端
- 具體案例：AMAT 收購 NEXX（面板 ECD）、TEL 熊本 R&D 投入（RDL 設備）、Lam Research（奈米晶銅 CMP）

**PLP 均一性挑戰量化**：510×515mm 有機面板 vs. 310×310mm 玻璃面板 → 均一性要求隨基板尺寸平方增長，需「封閉循環」製程系統。

**永續性量化驅動力**：化學品用量削減、廢水回收是 OSAT 進入 HVM 的成本競爭關鍵。
