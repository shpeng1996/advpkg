# Besi 站上封裝權力轉移核心：混合接合設備與 Kinex 平台

## 核心主張 / Key Claims

1. **Besi** 是 D2W（晶粒對晶圓）混合接合設備的主要供應商，與 EV Group（W2W 為主）形成互補
2. **Applied Materials** 持有 Besi **9% 股份**，並共同開發 **Kinex 平台**（全整合 D2W 混合接合解決方案）
3. Kinex 平台已接近**量產製造（HVM）就緒**狀態（截至 2026-03）
4. Besi 旗艦設備 **Datacon 8800 CHAMEO ultra plus AC** 目標：10 µm 以下接合間距
5. Die attach 約佔 Besi 2025 年營收的 **80%**，混合接合業務持續增長

## 關鍵數據 / Key Data Points

| 項目 | 數值/狀態 |
|------|---------|
| Applied Materials 持股 | 9% |
| 2025 die attach 營收佔比 | ~80% |
| 混合接合設備型號 | Datacon 8800 CHAMEO ultra plus AC |
| 目標接合間距 | <10 µm (hybrid bonding) |
| Kinex 平台狀態 | HVM 就緒（2026-03） |
| 混合接合研討會 | 2026-01-22/23，Milpitas, CA |
| 2023 大單 | 26 套混合接合系統訂單 |

## 新增知識 / New Knowledge Added

- **Besi** 作為獨立設備廠商尚無 wiki 實體頁面；本來源提供建立頁面的核心資料
- **Applied Materials + Besi Kinex 平台** 的合作關係是設備生態系中的重要連結
- D2W vs W2W 的設備商分工（Besi vs EV Group）是混合接合技術頁面需補充的關鍵架構
- **Fluxless TCB（Thermocompression Bonding）**作為<10µm 的替代路徑，在 Besi 演講中被提及

## 矛盾或修正 / Contradictions / Corrections

- EV Group 既有頁面著重 W2W；Besi 的 D2W 定位無矛盾，是互補關係
