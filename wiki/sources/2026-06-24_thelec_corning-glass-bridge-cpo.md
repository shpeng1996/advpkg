# Corning Glass Bridge — CPO 與玻璃核心封裝（2026-06-24）

## 核心主張 / Key Claims

1. Corning 推出 **GlassWorks AI Solutions** 平台，整合光纖、Cable、FAU、連接器到玻璃光學耦合的 AI 資料中心全棧光學解決方案。
2. **Glass Bridge** 是 Corning 新一代玻璃光學連接器，利用離子交換波導技術（wafer-based ion-exchange waveguide），解決 PIC 晶片（奈米級波導）與光纖（微米級）之間的尺寸不匹配問題（縮放比 ~數十倍）。
3. Corning 還展示了**結合玻璃基板＋TGV＋CPO 的新型封裝架構**，在玻璃基板上形成光學波導並整合覆晶光子裝置，為 CPO 提供全新封裝路線。
4. Corning 已與 GlobalFoundries 合作光學互連技術，並與 Meta、Nvidia、Amazon 簽訂多年期供應合約（數十億美元規模）。

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| Glass Bridge 支援 PIC 核心間距 | ≥30 µm（初始產品） |
| 目標光纖→光子晶片耦合損耗 | <2 dB |
| 技術原理 | 離子交換波導（wafer-based ion-exchange） |
| 玻璃基板 CPO 架構 | TGV 玻璃基板＋覆晶光子裝置＋光波導 |

## 新增知識 / New Knowledge Added

- **首次收錄 Corning Glass Bridge 產品規格**（30µm 核心間距、<2dB 耦合損耗、離子交換波導技術）——既有 wiki 中提及 Corning 主要作為玻璃基板材料供應商（BOE-Corning MOU 記錄），Glass Bridge 作為 CPO 主動器件的角色是全新資訊。
- **玻璃基板＋TGV＋CPO 整合架構**：Corning 的玻璃基板 CPO 展示確認玻璃基板不僅替代矽中介層，還可整合光學波導，是玻璃基板技術的顯著功能延伸。
- 與 GlobalFoundries 合作是對 wiki 中「GF+Corning CPO」記錄（ECTC 2026 半導體生態）的具體化補充。
