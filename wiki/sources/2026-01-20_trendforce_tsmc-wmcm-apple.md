# TSMC WMCM 產能 2027 年倍增（2026-01-20）

## 核心主張 / Key Claims

- TSMC WMCM（晶圓級多晶片模組）產能預計 **2027 年較 2025/26 基準倍增以上**
- 主要擴產設施：**AP3**（既有廠升級）＋ **AP7 Phase 2**（嘉義，Apple 專屬）
- 目標：**2026 年底 WMCM 產能達 60,000 片晶圓/月**
- Apple 是 WMCM 唯一大客戶（近期排他），驅動整個擴產計畫
- A20（iPhone 18，2026）→ A20 Pro/Ultra → M5 Pro/Max 均預計採用 WMCM
- InFO 持續保留，服務低階 Apple 晶片及其他客戶（過渡期並行）

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| WMCM 目標月產能（2026 年底） | 60,000 片晶圓 |
| 2027 倍增後估計 | >120,000 片晶圓 |
| 擴產廠房 | AP3（升級）+ AP7 Phase 2（Apple 專屬） |
| Apple WMCM 排他窗口 | 預計 2027–2028 開放給其他客戶 |

## 新增知識 / New Knowledge Added

- **60K wsm/月目標（2026 年底）**：具體 WMCM 產能數字首次在 wiki 記錄
- **AP7 Phase 2 Apple 專屬**：確認 AP7 第二期為 Apple WMCM 保留，補充了 AP7 技術布局的客戶維度
- **WMCM 排他窗口**：2027–2028 後開放其他客戶——意味著 WMCM 可能從 Apple 專屬技術演變為平台性封裝技術（類似 InFO 的發展路徑）
