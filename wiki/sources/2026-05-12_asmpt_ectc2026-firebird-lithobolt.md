# ASMPT ECTC 2026 展出 FIREBIRD TCB 與 LITHOBOLT D2W（官方新聞稿，2026-05-12）

## 核心主張 / Key Claims

1. **FIREBIRD 系列 TCB 平台**：支援 16 層 HBM 堆疊；Active Oxide Removal (AOR) 技術實現無助焊劑接合；全球已部署 >500 套 TCB 系統進入量產。
2. **LITHOBOLT™ G2 D2W 接合平台**：支援 >10,000 互連/mm² 的超高密度 Cu-Cu 接合，為 3D 整合提供無污染環境的無空洞接合。
3. ASMPT 定位為 TCB（高層 HBM）和 D2W 混合接合（邏輯-記憶體 3D 整合）雙平台領導商。

## 關鍵數據 / Key Data Points

| 規格 | 數值 |
|------|------|
| FIREBIRD 擺放精度 | 2.0 µm |
| FIREBIRD 已部署系統數 | >500 套（量產） |
| FIREBIRD 支援最高 HBM | 16 層（16H） |
| LITHOBOLT 互連密度 | >10,000 互連/mm² |
| LITHOBOLT 技術 | 無空洞 Cu-Cu；高精度 overlay |

## 新增知識 / New Knowledge Added

- **ASMPT FIREBIRD 系列具體規格首次記錄**：wiki 現有 ASMPT 頁面提及「TCB/混合接合/Photonics-CPO 設備領導廠商；Q1 2026 訂單 27M +71.6% YoY；NEXX 出售給 AMAT」，但缺少 FIREBIRD/LITHOBOLT 的具體技術規格。
- **AOR（Active Oxide Removal）技術**：ASMPT 的 TCB 差異化技術（無助焊劑），wiki 首次記錄。
- **>500 套 TCB 系統已在量產中部署**：量化了 ASMPT TCB 市場保有量規模。

## 矛盾或修正 / Contradictions / Corrections

- 無矛盾。LITHOBOLT D2W 與 D2W 混合接合（AMAT+Besi Kinex 系統）是不同廠商的平行解決方案，共存於市場。
