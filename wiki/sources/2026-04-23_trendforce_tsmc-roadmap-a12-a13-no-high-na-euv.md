# TSMC A12/A13 for 2029 Without High-NA EUV; A16 Delayed to 2027

**來源類型 / Source Type**：news
**原始路徑 / Original Path**：raw/articles/2026-04-23_trendforce_tsmc-roadmap-a12-a13-2029-no-high-na-euv.md
**URL**：https://www.trendforce.com/news/2026/04/23/news-tsmc-unveils-latest-roadmap-a12-a13-set-for-2029-without-high-na-euv-a16-volume-production-delayed-to-2027/
**發布 / Publisher**：TrendForce
**日期 / Date**：2026-04-23

## 核心主張 / Key Claims

1. **A13（1.3nm 級）宣布 2029**：A14 的光學縮小版，線性尺寸縮小 ~3%（≈97%），電晶體密度提升 ~6%；無需 High-NA EUV
2. **A12（1.2nm 級）同樣宣布 2029**：「激進的時程加速」，與 A13 同年，無需 High-NA EUV；路線圖壓縮意義重大
3. **A16 量產延至 2027**：原定 2026 年底量產，正式確認延期至 2027
4. **TSMC 明確不採用 High-NA EUV**：A12、A13 均依賴現有 Low-NA EUV 最佳化，非 High-NA；對 ASML 收入模型產生重大衝擊
5. **ASML 因此「靜悄悄轉向」**：High-NA 收入主要依賴 Intel 和 Samsung；ASML 原預期 High-NA 2027–28 大規模量產、2030 年創造 €600 億收入，現面臨修正

## 關鍵數據 / Key Data Points

| 節點 | 目標時程 | 是否需要 High-NA EUV |
|------|----------|---------------------|
| A14 | 2026–2027 量產 | 否 |
| A16 | **2027**（修正自 2026 年底） | 視 Intel 版本而定 |
| A13 | **2029**（新宣布） | 否 |
| A12 | **2029**（新宣布） | 否 |
| ASML 2026 銷售目標 | €360–400 億 | Low-NA 依然強勁 |
| ASML 韓國份額 | 45% | 三星 + SK Hynix 需求帶動 |

## 新增知識 / New Knowledge Added

- A13 的具體技術定義（A14 光學縮小 ~3%）和密度提升量化（~6%）首次揭露
- TSMC A12 + A13 同年（2029）的路線圖壓縮策略及其對 High-NA EUV 採用的影響
- ASML 的 High-NA 收入重新依賴 Intel + Samsung（非 TSMC）的市場現實
- ASML 調高 2026 銷售目標至 €36–40B 反映 Low-NA 訂單強勁
