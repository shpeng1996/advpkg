# Intel EMIB 潛在數十億美元營收（2H26）/ Intel EMIB Could Generate Billions by 2H26

## 核心主張 / Key Claims

- Intel CFO 表示 EMIB 封裝技術最早可於 **2026 年下半年**開始產生**數十億美元**營收
- 客戶名單包括 **Apple**（探索 EMIB 用於未來產品）、**Qualcomm**（尋求 chiplet 整合）、多家 AI ASIC 廠商
- Intel 正考慮重新開放 **18A 製程**供外部晶圓代工客戶使用
- EMIB 可封裝來自不同晶圓廠（包含 TSMC）的 die，形成跨晶圓廠異質封裝服務模式
- EMIB-T 目標封裝尺寸 **120×120mm**，超越 NVIDIA 100×100mm，可容納更大多晶片配置

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| 預期 EMIB 營收時程 | 2H26 開始 |
| 預期規模 | 數十億美元/年（近尾聲談判） |
| 目標封裝尺寸 | 120×120 mm（EMIB-T） |
| 競爭基準 | NVIDIA 採用 100×100 mm |

## 新增知識 / New Knowledge Added

- **「封裝即服務」商業模式**：Intel Foundry 即使邏輯製造業務不振，仍可透過 EMIB 封裝服務建立獨立營收流
- Intel EMIB 可封裝 TSMC-made dies → 兩大巨頭並非純競爭，存在互補可能
- Apple/Qualcomm 探索 EMIB 的具體細節——補充現有 intel.md 中客戶名單
- 18A 外部代工重新考慮：Intel 策略轉向，封裝為切入點
