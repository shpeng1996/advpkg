# SK 會長 COMPUTEX 2026：記憶體短缺持續至 2030 / SK Chair at Computex 2026: Memory Shortage Through 2030

## 核心主張 / Key Claims

- SK Group 會長崔泰源（Chey Tae-won）首次出席 COMPUTEX（6/2），預測**記憶體供需吃緊將持續至 2030 年**。
- SK Group 目標：**5 年內晶圓總產能翻倍**（在現有供給吃緊下全力擴產）。
- NVIDIA CEO Jensen Huang 連續兩天與 SK 會長互動（前一晚晚餐 + 6/2 展場巡視 SK hynix 攤位）；Huang 在前一天 Keynote 親口點名 SK hynix 為 next-gen HBM4 供應商。
- **HBM4E 實體 mock-up 首次公開亮相**（Chosun Biz：SK hynix booth，6/2）；HBM4E wafer + chipset 樣品已於 5 月底開始 sampling。
- SK hynix HBM4 技術細節確認：
  - HBM4（NVIDIA 2026 年供貨）：**1b DRAM 核心 die + TSMC 12nm logic die**
  - HBM4E：**1c DRAM 核心 die + TSMC 3nm logic die**

## 關鍵數據 / Key Data Points

| 項目 | 數值 | 說明 |
|------|------|------|
| 記憶體短缺預測期 | 至 2030 | SK Chair 公開聲明 |
| 產能擴充目標 | 晶圓總產能 ×2（5 年） | SK Group 目標 |
| HBM4 core die | 1b（10nm 5th gen） | SK hynix 確認 |
| HBM4 logic die | TSMC 12nm | SK hynix 確認 |

| HBM4E sampling 時間 | 2026 年 5 月底 | Chosun Biz |

## 新增知識 / New Knowledge Added

- **HBM4 / HBM4E 製程堆疊細節首次完整確認**：wiki 先前 `hbm4.md` 記錄 SK hynix 使用 TSMC logic die，但 1b/1c DRAM 製程節點 + TSMC 12nm/3nm 組合是首次在公開場合正式確認。這對了解 TSMC logic die 的需求量化至關重要。
- **記憶體短缺至 2030 的高層預測**：為現有 wiki 市場概念頁面增添具體決策依據（此前預測多為分析師；此次為 SK Group 最高決策者公開表態）。
- **HBM4E 實體 mock-up 首次亮相（6/2 COMPUTEX）**：時間節點記錄。

## 矛盾或修正 / Contradictions / Corrections

- 無矛盾；補充技術細節（1b/1c + 12nm/3nm 組合）與市場展望（至 2030）。
