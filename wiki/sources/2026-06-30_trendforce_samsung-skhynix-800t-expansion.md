# 三星+SK Hynix KRW 800T 擴產壓縮設備供應鏈 / Samsung+SK hynix KRW 800T Expansion Strains Equipment Supply

## 核心主張 / Key Claims

- 三星與 SK hynix 合計宣布 KRW 800T 擴產計畫，包含 4 座新廠與 KRW 81T 先進封裝中心（忠清）
- 同步大規模建廠造成 EUV、蝕刻、CMP、沉積等關鍵設備供應吃緊
- 設備交期延誤恐波及台積電與 Intel 的 sub-2nm 製程推進速度
- SK hynix 龍仁 4 廠時程提前 12 年（2045→2033）
- 三星 P5+P6 雙廠同步建設於平澤
- 南亞科技轉型利基市場：NT$52B 資本支出、1D DDR5 試產（Q2 2026）、HBM 探索中

## 關鍵數據 / Key Data Points

| 項目 | 數據 |
|------|------|
| 合計投資 | KRW 800T |
| 先進封裝中心 | KRW 81T（忠清，SK hynix P&T7） |
| SK hynix 龍仁 4 廠 | 時程提前 12 年（2045→2033） |
| 三星 Pyeongtaek | P5+P6 同步建設 |
| 南亞科技資本支出 | NT$52B |
| 南亞 DDR5 試產 | Q2 2026（1D 製程） |

## 新增知識 / New Knowledge Added

- KRW 81T 先進封裝中心是半導體史上最大單筆封裝基礎設施投資宣告
- 韓廠大規模擴產的「設備排擠效應」作為衝擊台積電/Intel 節奏的二階影響首次被量化分析
- 南亞科技 HBM 探索為台灣記憶體廠商切入先進封裝市場的潛在訊號

## 矛盾或修正 / Contradictions / Corrections

800T 為 Samsung+SK hynix 合計之更高層數字，與 2026-07-02 TrendForce 報導（SK hynix 100T + Samsung 140T 子公司明細）需搭配閱讀。兩文一致，無矛盾。
