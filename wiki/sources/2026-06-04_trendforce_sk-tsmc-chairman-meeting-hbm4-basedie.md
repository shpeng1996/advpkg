# SK 集團會長與台積電董事長兩年來首度會面：HBM4 base die 委外台積電

## 核心主張 / Key Claims

1. **高層會面釋出深化合作信號**：SK 集團會長崔泰源與台積電董事長魏哲家於 2026 年 6 月 4 日會面——兩人時隔兩年首次見面，會談聚焦 HBM 與先進封裝合作，顯示兩家公司在 AI 記憶體供應鏈上的關係正在升溫。

2. **HBM4 base die 生產模式結構性轉變**：SK hynix 過去自行生產 HBM 的 base die（基底邏輯晶片），但從 HBM4 世代開始，已將 base die 委外給台積電（採用台積電 12nm 製程生產），標誌著 SK hynix 記憶體製造模式的重大轉折。

3. **HBM4E 世代可能進一步深化合作**：SK hynix 正評估在 HBM4E 世代採用台積電 3nm 製程生產 base/logic die，目的是對抗三星憑藉自有 4nm 製程自製 base die 的競爭優勢。

4. **CoWoS 產能吃緊持續**：重申 2026 年底台積電 CoWoS 產能將達 115K–140K wsm（千片晶圓／月），2027 年將擴增到約 170K wsm，顯示先進封裝產能瓶頸短期內不會緩解。

5. **SK hynix 同步探索 Intel EMIB 作為替代封裝方案**：顯示 SK hynix 正在進行封裝供應鏈多元化布局，不完全依賴台積電 CoWoS。

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| HBM4 base die 製程（委外台積電） | 12nm |
| HBM4E base die 評估製程（台積電） | 3nm |
| Samsung 自製 base die 製程 | 4nm |
| 台積電 CoWoS 產能（2026 年底預估） | 115K–140K wsm |
| 台積電 CoWoS 產能（2027 年預估） | ~170K wsm |
