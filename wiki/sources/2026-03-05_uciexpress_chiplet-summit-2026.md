# Chiplet Summit 2026：UCIe 生態系動能 / Chiplet Summit 2026: UCIe Momentum

## 核心主張 / Key Claims

- Chiplet Summit 2026 (2026 年 3 月) 是 UCIe 由「規格」轉向「量產矽」的重要里程碑
- **業界首次 UCIe-S 跨廠商互操作性現場示範**：Intel + Cadence「Cameron Creek」測試晶片，驗證 16G UCIe-S PHY IP 跨供應商互操作性
- UCIe 3.0 榮獲 **Chiplet Summit Best in Show（連接性與互操作性類）**獎項
- UCIe Consortium 主席 Debendra Das Sharma 強調 UCIe 為**多十年產業標準**，從 1.0（平面）→ 2.0（3D 垂直）→ 3.0（雙倍頻寬密度+執行期功耗+管理能力）的完整演進路線
- Intel 與 Google 在 Tutorial 中提供 UCIe 實際部署指導（光學 chiplet、可組合 SoC、多裸片架構）

## 關鍵數據 / Key Data Points

- Chiplet Summit 2026 生態系參與廠商（完整清單）：
  - 贊助方：Arm、Marvell、Cadence、Synopsys、Siemens、Eliyan
  - 展覽方：Alphawave Semi、Cadence、Siemens、Synopsys、Keysight Technologies、Tenstorrent、Menta、VeriSilicon、proteanTecs、InPsytech、Extoll
- Cameron Creek 測試晶片：Intel + Cadence 共同設計，16G UCIe-S PHY IP 首次跨廠商驗證
- UCIe 3.0 倍增平面頻寬密度（詳見 2025-09-03 規格書）

## 新增知識 / New Knowledge Added

- **Cameron Creek 跨廠商互操作示範**是 UCIe 從規格到矽的關鍵轉折點，wiki 現有 UCIe 頁面缺少此重要里程碑
- Eliyan 出現在贊助名單（新玩家），是高頻寬短距 die-to-die 互連 IP 廠商，可考慮建立實體頁面
- Tutorial 中提及「光學 chiplet 部署 UCIe」，與 wiki 的 TSMC COUPE CPO 頁面有交叉，確認 UCIe 在光電整合中的角色
