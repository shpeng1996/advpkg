# SK hynix HBM4E 樣品時程提前 / SK hynix Pulls Forward HBM4E Sample Timeline

## 核心主張 / Key Claims

- SK hynix 將 HBM4E 樣品出貨時程提前，目標 2026 年 6–7 月對主要客戶交付樣品，早於市場原先預期。
- Samsung 競爭規格鎖定 1c+4nm 製程組合，目標 14–16 Gbps、約 4TB/s/stack。
- SK hynix HBM4E 預計搭配 1c DRAM core + 台積電 3nm logic base die，延續台積電代工 base die 模式。
- NVIDIA Rubin Ultra 平台明年 HBM 容量目標每 GPU 384GB，是時程提前的需求背景。

## 關鍵數據 / Key Data Points

- 樣品時程：2026-06 ~ 2026-07
- Samsung HBM4E：1c+4nm、14–16 Gbps、~4 TB/s
- NVIDIA Rubin Ultra：384GB HBM / GPU（明年）
