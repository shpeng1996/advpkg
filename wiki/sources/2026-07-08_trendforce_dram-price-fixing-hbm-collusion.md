# DRAM Price-Fixing Case — HBM Expansion as Collusion Evidence / TrendForce 2026-07-08

## 核心主張 / Key Claims

1. Samsung, SK hynix, Micron 在美國面對新一輪集體訴訟（14 名消費者 + 3 家小型 PC 廠商），指控三者協調削減傳統 DRAM 產量，推動記憶體價格 4 年內上漲近 700%
2. 此次訴訟的核心新穎性：原告嘗試以**HBM 擴產**作為協調削減 DRAM 供應的直接證據——先前此類訴訟未曾使用 HBM 轉型作為論據
3. 法律專家認為，單純的價格同步上漲不足以證明共謀；原告需提供更具體的協調證據（如共享敏感定價資料、在業界會議協商策略等）
4. 三大廠商的反制論點：AI 伺服器需求飆升為記憶體漲價提供正當性；龐大的產能擴張計畫（Micron 廣島、SK hynix 龍仁、Samsung 新廠）與刻意限產論點矛盾
5. 歷史判例：2018 年類似集體訴訟（指控 2016–2018 年供應協調）於 2022 年被法院駁回，因證據不足

## 關鍵數據 / Key Data Points

| 指標 | 數值 |
|------|------|
| 記憶體價格漲幅（4年） | ~700% |
| 三大廠商 DRAM 市占 | ~90% |
| Samsung 2005 年反壟斷罰款 | $3 億美元（當時美國史上第二高） |
| 2022 年前次訴訟結果 | 聯邦地院駁回（證據不足） |

## 新增知識 / New Knowledge Added

- 首次記錄 HBM 轉型被用於反壟斷訴訟的法律框架分析
- 確認「HBM 擴產 + commodity DRAM 削減」的敘事風險已進入法律層面
- 若訴訟存活，三大廠商可能首次被迫揭露 HBM 配額分配的內部通訊
