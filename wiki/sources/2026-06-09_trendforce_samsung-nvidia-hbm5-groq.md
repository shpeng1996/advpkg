# 三星與 NVIDIA 深化合作：HBM5、Groq LP40 / Samsung-NVIDIA Deepen Ties

## 核心主張 / Key Claims

- 三星 DS 事業部 Jun Young-hyun 副會長與黃仁勳於 2026-06-08 會面，討論 HBM4、SOCAMM2 近期穩定供應及明年起 HBM4E、代工、HBM5 長期合作
- 三星正以 4nm + 8nm 製造 NVIDIA 自動駕駛芯片（Drive AGX Thor）及 Groq LPU（LP30）；Jun 確認討論下一代 Groq LP40 合作
- 三星 HBM4 數據速率：**>11.7 Gbps/pin**；HBM4E 以自家 4nm base die 製造，速率 **14 Gbps/pin**（測試可達 16 Gbps）
- 三星同時供應 Vera CPU 的 LPDDR5X SOCAMM2 模組及 PCIe Gen6 PM1763 儲存方案

## 關鍵數據 / Key Data Points

| 產品 | 規格 | 製造節點 |
|------|------|---------|
| Samsung HBM4 | >11.7 Gbps/pin | TSMC 12nm base die |
| Samsung HBM4E | 14 Gbps/pin (tested 16 Gbps) | Samsung 4nm base die |
| Groq LP30 LPU | 量產中 | Samsung 4nm |
| Groq LP40 LPU | 討論中 | Samsung（潛在競爭 TSMC） |
| SOCAMM2 | LPDDR5X | Vera CPU 供貨中 |

## 新增知識 / New Knowledge Added

- **Samsung HBM4 具體 I/O 速率首次入庫**：>11.7 Gbps/pin（確認 JEDEC 11.7 Gbps 基準以上）
- **Samsung HBM4E 明確採用自家 4nm base die**（vs SK Hynix 用 TSMC 12nm / HBM4E 評估 TSMC 3nm），顯示技術路線分歧
- **Groq LP40 合作討論首次揭露**：先前 wiki 僅記錄 Samsung 製造 Groq LP30（4nm）；現確認下一代 LP40 討論中
- **三星在 Vera 平台的記憶體組合完整化**：HBM4（GPU）+ SOCAMM2（CPU）+ PCIe Gen6 SSD（完整 AI 基礎架構覆蓋）

## 矛盾或修正 / Contradictions / Corrections

- 此前 wiki 記錄 Samsung HBM4E base die 採「自家 4nm」，本文進一步確認（14/16 Gbps 速率為內部測試值，與先前的 14-16 Gbps 官方規格一致，但第三方驗證仍待確認）
